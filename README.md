# Supermarket Management System

![Supermarket]

This project is a simple Java application for managing a supermarket's product inventory. It provides basic functionality for adding, modifying, deleting, and searching for products in a supermarket.

## Features

- **Insert Product**: Add new products to the supermarket inventory.
- **Modify Product**: Update existing products in the inventory.
- **Delete Product**: Remove products from the inventory.
- **Find Product**: Search for a product in the inventory.
- **Copy Inventory**: Create a copy of the supermarket's product inventory.
- **Display Inventory**: View the current list of products in the supermarket.
- **Menu System**: User-friendly menu for easy navigation.
